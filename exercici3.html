CREATE TABLE Examples

Basic CREATE TABLE statement

CREATE TABLE stats (
  id INT,
  name VARCHAR(255),
  score INT
);

The CREATE TABLE statement

The CREATE TABLE statement is used to create a new table in a database.

Syntax:

CREATE TABLE table_name (
  column1 datatype,
  column2 datatype,
  ...
);

Creating a table with a particular storage engine

MyISAM provides a number of differences with InnoDB. The notable and most notable difference is that InnoDB supports the use of transactions and row-level locking, while MyISAM supports only table-level locking.

CREATE TABLE stats (
  id INT,
  name VARCHAR(255),
  score INT
) ENGINE=MyISAM;

Creating a table with auto increment

This can be useful to automatically assign unique identifiers to each new row.

CREATE TABLE stats (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(255),
  score INT
);

Creating a table with the current timestamp

You can use the NOW() function to insert the current timestamp into a new row. The NOW() function returns the current date and time. The data type for the timestamp column should be TIMESTAMP.

CREATE TABLE stats (
  id INT AUTO_INCREMENT
